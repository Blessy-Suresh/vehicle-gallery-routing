**Vehicle Gallery Routing**

This is a React + Vite project that demonstrates routing and component navigation.

**Features**
- React Router implementation
- Navigation using buttons
- Vehicle categories:
  - Bike
  - Car
  - Truck
- Modal display for images
- Clean UI layout


**Project Structure**

```
vehicle-gallery-routing/
│
├── src/
│   ├── vechicle/
│   │   ├── Bike.jsx
│   │   ├── Car.jsx
│   │   └── Truck.jsx
│   │
│   ├── App.jsx
│   └── main.jsx
│
├── index.html
├── package.json
├── vite.config.js
└── README.md
```
